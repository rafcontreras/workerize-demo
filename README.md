# workerize-demo

- Clone the repo

```bash
git clone git@github.com:rafcontreras/workerize-demo.git
```

- Move to directory

```bash
cd workerize-demo
```

- Install npm packages

```bash
yarn
```

or

```bash
npm i
```

- Run the demo locally

```bash
yarn develop
```

or

```bash
npm run develop
```
